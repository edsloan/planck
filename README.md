# planck

v1.0.0

The worlds smallest CSS grid system

## How small?

It's 284 bytes minified and 168 bytes gzipped (minus comments).

## Usage

- Include `planck.min.css` within the `<head>` of your HTML:
	
```html
	<head>
		<link rel="stylesheet" href="planck.min.css">
	</head>
```

- Example structure:

```html
	<div class="x6">
		<!--full width-->
	</div>
	<div class="x2">
		<!--one-third column-->
	</div>
	<div class="x2">
		<!--one-third column-->
	</div>
	<div class="x2">
		<!--one-third column-->
	</div>
```
